Style Stage

Style Stage is a project that I completed for my Advanced CSS class. We were given the HTML code for a web page and told to customize it to fit whatever theme we would like. The only rule was that we couldn’t change the HTML or use JavaScript. This made it a little tricky because I had to find ways to move and customize parts of the page while only using SCSS. Ultimately, my partner and I created this retro-themed version of the Style Stage website. You are welcome to check out the original website and see how ours differs.
